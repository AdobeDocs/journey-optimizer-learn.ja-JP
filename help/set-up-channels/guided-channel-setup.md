# ガイド付きチャネル設定

ガイド付きチャネルの設定は、Experience Platform、Journey Optimizer、データ収集をまたいで必要なリソースを使用して、web チャネルとモバイルチャネルを迅速に設定および検証し、マーケティングチームがキャンペーンとジャーニーの作成を開始できるようにします。
