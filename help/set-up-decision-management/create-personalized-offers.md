# パーソナライズされたオファーの作成

[!DNL Offer Decisioning] でパーソナライズされたオファーを作成する方法を説明します。 パーソナライズされたオファーには実施要件ルールが関連付けられ、関連する顧客のみに表示するのに役立ちます。 例えば、雪のショベルのオファーを、雪が降る地域と冬の間に限定することができます。
