# タグの作成

方法を学ぶ [タグを作成](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-components/creating-tags.html?lang=ja) in [!DNL Offer Decisioning]. タグは、オファーのオプションの構成要素です。これらを使用すると、オファーを整理して、動的なコレクションにグループ化できます。
