# オファーアクティビティの作成

オファーアクティビティの作成方法を [!DNL Offer Decisioning] において説明します。オファーアクティビティは、プレースメントとコレクションを 1 つのエンティティに組み合わせたものなので、オファーアクティビティを使用すると、最も関連性の高いオファーを顧客に提供するように決定できます。
