# 決定 API を使用したオファーの配信

決定 API を使用して [!DNL Offer Decisioning] のオファーを配信する方法を説明します。[!DNL Postman] を使用して、API 呼び出しによるコンテナ ID の取得のデモを行います。次に、アクティビティ ID、プレースメント ID、ID 名前空間および顧客 ID でサンプルの決定 API 呼び出しをカスタマイズして、パーソナライズされたオファーを取得します。パーソナライズされたオファーを決定 API で取得したら、そのオファーコンテンツをマーケティングシステムに統合できます。
