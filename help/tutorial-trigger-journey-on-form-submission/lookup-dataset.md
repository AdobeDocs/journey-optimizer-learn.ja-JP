# ルックアップデータセットの作成

データセットルックアップを使用すると、Adobe Journey Optimizerは、ジャーニー実行時にAdobe Experience Platform データセットから参照データやトランザクションデータを取得できます。そのデータをリアルタイム顧客プロファイルに保存する必要はありません。

このユースケースでは、ユーザーが投資環境設定フォーム（株式、債券、または CD）を送信すると、ジャーニーは参照データセットから対応する投資詳細を動的に検索し、その情報を使用してメールをパーソナライズします。
