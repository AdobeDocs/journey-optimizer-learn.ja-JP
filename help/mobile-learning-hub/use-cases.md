# ユースケース

| 使用例 | 説明 | 使用例 |
|---------|-------------|----------------------|
| **アプリのオンボーディングと導入** | アプリのインストール、設定の完了、主要機能の検索など、アプリのエンゲージメントの初期段階で新規ユーザーを導きます。 リテンションと長期的な利用を最大化することが目標です。 | - プッシュ通知とSMSは、ユーザーを歓迎し、プロファイルの完了を促します。<br>- アプリ内メッセージは、機能をハイライト表示し、最初のアクションを促します。<br>- メールまたは広告のディープリンクは、ユーザーを特定のアプリ画面に誘導して、シームレスなオンボーディングを実現します。 |
| **場所ベースのエンゲージメント** | 店舗やイベント、その他の関連する場所への物理的な近さに基づいて、ユーザーにパーソナライズされたタイムリーなメッセージを提供します。 | - ジオフェンシングとビーコンのテクノロジートリガーのプッシュ通知を利用して、ユーザーがターゲットゾーンに入った場合に通知します。<br>- SMS/MMSは、ローカライズされたオファーと更新を配信します。<br>- アプリ内バナーとカードは、リアルタイムの場所に基づいてコンテンツを適応させます。 |
| **放棄のリエンゲージメント** | カートやフォーム、閲覧セッションを放棄した利用者をターゲットとし、再訪問を促して意図したアクションを完了することを目指します。 | - プッシュ通知は、カート放棄または不完全なアクションをユーザーに通知します。<br>- SMS フォローアップには、再開するためのインセンティブまたは直接リンクが含まれます。<br>- ユーザーが戻ると、アプリ内プロンプトが表示され、パーソナライズされたレコメンデーションが提供されます。 |
| **アップセルとクロスセルのキャンペーン** | 顧客の行動、好み、購買履歴にもとづいて、既存顧客に追加製品のプロモーションやアップグレードを提供。 | サブスクリプション登録と売上増加を促進 |
| **解約防止** | 離脱するリスクのある利用者を特定し、パーソナライズされた維持戦略でエンゲージしてロイヤルティを維持します。 | - Predictive Analytics トリガーのモバイル アウトリーチをリスクのあるユーザーに提供します。<br>- プッシュ通知とSMSは、ロイヤルティ報酬またはパーソナライズされたコンテンツを提供します。<br>- アプリ内の調査では、リテンション戦略を改善するためにフィードバックを収集します。 |
| **マルチチャネル メッセージ** | 複数のモバイルチャネルをまたいで一貫性のあるメッセージを配信し、オーディエンスがタイムリーで関連性の高いコミュニケーションを受け取れるようにします。 | - プッシュ、アプリ内、SMS、および電子メールは、統合されたメッセージ用に調整されます。<br>- SDKにより、チャネルをまたいだリアルタイムのパーソナライゼーションが可能になります。<br>- コンテンツカードはセッション間で保持され、主要なメッセージを強化します。 |
