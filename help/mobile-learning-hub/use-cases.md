# ユースケース

| 使用例 | 説明 | 使用例 |
|---------|-------------|----------------------|
| **アプリのオンボーディングと導入** | アプリのインストール、設定の完了、主要機能の検索など、アプリのエンゲージメントの初期段階で新規ユーザーを導きます。 リテンションと長期的な利用を最大化することが目標です。 | - プッシュ通知とSMS ウェルカムユーザーとプロンプトプロファイルの完了。<br>- アプリ内メッセージは、機能を強調し、最初のアクションを促します。<br>- メールまたは広告内のディープリンクを使用して、ユーザーを特定のアプリ画面に誘導し、シームレスなオンボーディングを実現します。 |
| **場所ベースのエンゲージメント** | 店舗やイベント、その他の関連する場所への物理的な近さに基づいて、ユーザーにパーソナライズされたタイムリーなメッセージを提供します。 | - ジオフェンシングとビーコン技術トリガーのプッシュ通知は、ユーザーがターゲットゾーンに入ったときに通知します。<br>- SMS/MMSは、ローカライズされたオファーと更新を配信します。<br>- アプリ内バナーとカードは、リアルタイムの場所に基づいてコンテンツを適応させます。 |
| **放棄のリエンゲージメント** | カートやフォーム、閲覧セッションを放棄した利用者をターゲットとし、再訪問を促して意図したアクションを完了することを目指します。 | - プッシュ通知は、放棄されたカートや不完全なアクションをユーザーに思い出させます。<br>- SMS フォローアップには、再開のインセンティブまたは直接リンクが含まれます。<br>- ユーザーが再訪問すると、アプリ内プロンプトが表示され、パーソナライズされたレコメンデーションが提供されます。 |
| **アップセルとクロスセルのキャンペーン** | 顧客の行動、好み、購買履歴にもとづいて、既存顧客に追加製品のプロモーションやアップグレードを提供。 | サブスクリプション登録と売上増加を促進 |
| **解約防止** | 離脱するリスクのある利用者を特定し、パーソナライズされた維持戦略でエンゲージしてロイヤルティを維持します。 | - リスクのあるユーザーへの予測分析トリガーモバイルアウトリーチ。<br>- プッシュ通知とSMSは、ロイヤルティ報酬またはパーソナライズされたコンテンツを提供します。<br>個のアプリ内アンケートでは、保持戦略を改善するためにフィードバックを収集します。 |
| **マルチチャネル メッセージ** | 複数のモバイルチャネルをまたいで一貫性のあるメッセージを配信し、オーディエンスがタイムリーで関連性の高いコミュニケーションを受け取れるようにします。 | - プッシュ、アプリ内、SMS、およびメールは、統合されたメッセージ用に調整されます。<br>- SDKにより、チャネルをまたいだリアルタイムのパーソナライゼーションが可能になります。<br>- コンテンツカードはセッション間で保持され、重要なメッセージを強化します。 |
