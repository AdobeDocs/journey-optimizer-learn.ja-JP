# レッスン 3 – 動的コンテンツの作成

*お客様の課題：* アプリをまだインストールしていないユーザーのみがインストールを求められることを確認する必要があります。 既にアプリをインストールしている人は、一般的なウェルカムメッセージのみを受け取る必要があります。 ジャーニーフローを複雑にしないでください。 

*解決策*：動的コンテンツを使用すると、プロファイル属性、コンテキストイベント、またはオーディエンスで構成できる条件付きルールに基づいて、メッセージのコンテンツを適応させることができます。 

この演習では、例としてSMSに動的コンテンツを適用しますが、すべてのチャネルに適用できます。

このトピックに関する詳細なガイダンスについては、[条件ルールビルダー](/help/personalize-content/create-dynamic-content.md)を使用した動的コンテンツの作成を参照してください。
