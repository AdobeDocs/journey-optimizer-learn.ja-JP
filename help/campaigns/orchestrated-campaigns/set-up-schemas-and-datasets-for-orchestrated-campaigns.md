# キャンペーンのスキーマとデータセットの設定

Adobe Journey Optimizerでオーケストレーションされたキャンペーンのリレーショナルスキーマとデータセットを設定する方法について説明します。 このガイドでは、キーの定義、関係の定義、DDL ファイルの使用による効率的なスキーマの作成について説明します。 また、CSV アップロードとChange Data Capture （CDC）を介したデータの取り込み、データセットの検証、クエリ、パーソナライゼーション、キャンペーン実行の準備についても説明します。 キャンペーンワークフローを効率化するための詳細な手順を、動画でご確認ください。
