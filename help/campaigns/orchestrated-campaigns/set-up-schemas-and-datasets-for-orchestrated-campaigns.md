# キャンペーンのスキーマとデータセットの設定

Adobe Journey Optimizer でオーケストレーションキャンペーンのリレーショナルスキーマとデータセットを設定する方法について説明します。 このガイドでは、キーと関係の定義、DDL ファイルを使用してスキーマを効率的に作成する方法について説明します。また、CSV アップロードやデータキャプチャを変更（CDC）を介したデータの取り込み、データセットの検証、クエリ、パーソナライゼーション、キャンペーン実行の準備についても説明します。キャンペーンワークフローを効率化する手順について詳しくは、組み込みのビデオをご覧ください。
