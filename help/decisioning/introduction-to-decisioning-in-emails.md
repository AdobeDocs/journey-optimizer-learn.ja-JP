# メール決定の概要

意思決定機能を利用して、キャンペーンの電子メールのパーソナライゼーションを変革する方法を紹介します。 このガイドでは、各受信者が最も関連性が高く、優先度の高いオファーを確実に受け取れるようにするための決定機能の使用に焦点を当てています。 たとえば、旅行業界では、顧客プロファイルにもとづいてオファーをインテリジェントに選択およびランク付けすることで、エンゲージメントと収益を最大化できます。

このプロセスには、オファーのコレクションを作成し、適格性ルールを割り当て、AI モデル、式、静的優先度スコアなどのランキング方法を適用することが含まれます。 マーケターは、上位オファーを効果的にメールに表示するように、選択戦略を調整できます。 テストを実施することで、戦略が正確な結果をもたらすことができます。

この強力なツールは、顧客エンゲージメントを強化し、インパクトのあるビジネス成果を促進します。 詳細なチュートリアルについては、組み込みビデオをご覧ください。
