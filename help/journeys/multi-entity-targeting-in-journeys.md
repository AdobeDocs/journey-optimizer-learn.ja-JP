# ジャーニーの連合オーディエンスを含むマルチエンティティターゲティングの使用

プロファイルベースのオーディエンスを超え、連合オーディエンスを使用してより関連性の高いカスタマーエクスペリエンスを提供するために、マルチエンティティターゲティングを使用する方法について説明します。

このビデオでは、サブスクリプション ID、アカウント ID、注文 ID 別にジャーニーをターゲットにする方法について説明します。これにより、1 人の顧客が複数のビジネスエンティティに関連付けられている場合に、より正確なオーディエンスアクティベーションとパーソナライズされたエンゲージメントが可能になります。

この機能について詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/journey-optimizer/using/orchestrate-journeys/manage-journey/supplemental-identifier){target="_blank"}を参照してください。
