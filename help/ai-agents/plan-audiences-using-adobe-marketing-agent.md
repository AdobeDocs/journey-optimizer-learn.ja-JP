# Adobe Marketing Agent for Microsoft 365 Copilotを使用したオーディエンスのプランニング

Adobe Marketing Agent for Microsoft 365 CopilotでMicrosoft Teams内のオーディエンス計画を簡素化する方法について説明します。 このツールを利用することで、プラットフォームを切り替えたり、運用チームにリクエストを送信したりすることなく、ロイヤルティの高いオーディエンスを特定し、ジャーニーの系統を確認し、キャンペーン候補を検証することができます。

このビデオでは、キャンペーン用のオーディエンスの選択、分析、準備のプロセスを説明します。 エージェントが実行可能なインサイトを生成し、関連する提案を表示する方法を紹介し、オーディエンスとジャーニー間のリネージュの可視化を提供します。 マーケターはデータを効率的に管理し、ワークフローを合理化して、集中したターゲティングとキャンペーンの成果を向上できます。

Adobe Marketing Agentの実際の動作を動画でご確認ください。
