# プロファイルの取り込みとオーディエンスの作成

このビデオでは、Adobe Experience Platformでプロファイルスキーマを作成し、CSV ファイルから顧客データを取り込み、プロファイル属性を使用してオーディエンスを構築する方法を説明します。

氏名、電子メール、ロイヤルティステータス、年収などの主要なフィールドを定義し、実際の顧客データをアップロードして、そのデータを使用して価値の高い顧客のオーディエンスを作成する方法を説明します。
